# 🎮 Tic Tac Toe Web Game

A classic Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This project demonstrates fundamental front-end development concepts including DOM manipulation, event handling, and game logic implementation.

---

## 🚀 Project Overview

This is a two-player interactive browser-based Tic Tac Toe game where players take turns marking X and O on a 3×3 grid. The game automatically detects the winner and prevents further moves once the game ends.

---

## ✨ Features

- 🎮 Two-player gameplay (X vs O)
- 🏆 Automatic winner detection
- 🔄 Reset / New Game functionality
- ❌ Prevents moves after game completion
- 🎨 Clean and responsive UI
- 📱 Mobile-friendly design
